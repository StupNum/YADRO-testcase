Компилировать с помощью MinGWw64 версией 7.3.0 и выше.
Команда для компиляции:  g++ --std=c++17 -o testcase main.cpp FileHandler.cpp GameClub.cpp Time.cpp common.cpp  
Запускать через консоль указывая вторым параметром файл (для Виндовс ./testcase.exe testinput.txt).
В репозитории находится файл testinput.txt содержащий тестовый ввод из ТЗ, а так же testout.txt, в нем содержится вывод программы при выполении с файлом testinput.txt,
он полностью соотвествует выводу из ТЗ.
